<h2>Wi-Fi QR Code Format</h2>

<code>
WIFI:T:WPA;S:mynetwork;P:mypasscode;;

</code>
<p>
<ul>
<il>WPA—the authentication type</il>
<il>mynetwork — the network SSID</il>
<il>mypasscode — the password</il>
<il>⚠️ Hidden SSIDs are not supported.</il>
<ul>
</p>
