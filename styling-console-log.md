# Styling console.log()

_Status: Published_
_Created: 2021-04-21 12:05:56_
_Tags: html, javascript_

<h2>Styling in console.log()</h2>
<code>
console.log("%c%s", "color: white; background-color: orange; font-style: italic;","my message")
</code>
<p>
%c - format styling</br>
%s - format string</br>
%o - dom object</br>
%O - Javascript object</br>
%i %f %d - numbers</br>
</p>
