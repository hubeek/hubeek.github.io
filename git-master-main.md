# Git master main

_Status: Published_
_Created: 2023-05-14 04:13:17_
_Tags: git, bash_

## Rename to main

```
git branch -m main 
```
push to origin
```
git push origin -u main
```

## Update config
```
git config --global init.defaultBranch main
```

## delete master
```
git branch -d master
```
